Link Manager 

Проект Link Manager предназначен для управления сохранёнными ссылками. Пользователи могут регистрироваться, входить в свои аккаунты и выходить из них. Все пароли пользователей хешируются для обеспечения безопасности. Данные хранятся в базе данных SQLite.

Функции
Регистрация аккаунта: Пользователи могут создать новый аккаунт.
Вход в аккаунт: Доступ к сохранённым ссылкам защищён паролем.
Выход из аккаунта: Пользователи могут выходить из своих аккаунтов.
Сохранение ссылок: Пользователи могут добавлять ссылки в свою коллекцию.
Удаление ссылок: Возможность удаления сохранённых ссылок.
Технологии
Языки программирования: Python
База данных: SQLite
Хеширование паролей: hashlib или bcrypt
Установка
Клонируйте репозиторий:
bash



Копировать код
pip install flask
Настройте базу данных: Убедитесь, что файл базы данных database.db создан в корне проекта.


Регистрация:
Введите желаемый логин и пароль для регистрации нового пользователя.

Вход в аккаунт:
После регистрации можно войти, используя логин и пароль.

Добавление ссылок:
После входа в систему, используйте интерфейс для добавления новых ссылок.

Удаление ссылок:
Выберите ссылку из списка и выполните команду удаления.
Безопасность:

Пароли пользователей хешируются с использованием алгоритма хеширования, чтобы защитить данные.
Обеспечивается безопасное соединение между клиентом и сервером (если это веб-приложение).# -
